at(position)

Returns the character at the position treating the string as an array (where 0 is the first character).

Examples:

"hello".at(0)  # => "h"
"hello".at(4)  # => "o"
"hello".at(10) # => nil

def last(limit = 1)
  (chars[(-limit)..-1] || self).to_s
end
