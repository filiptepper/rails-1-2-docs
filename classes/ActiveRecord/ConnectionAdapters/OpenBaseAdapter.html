The OpenBase adapter works with the Ruby/Openbase driver by Tetsuya Suzuki. www.spice-of-life.net/ruby-openbase/ (needs version 0.7.3+)

Options:

  • :host -- Defaults to localhost

  • :username -- Defaults to nothing

  • :password -- Defaults to nothing

  • :database -- The name of the database. No default, must be provided.

The OpenBase adapter will make use of OpenBaseâ€™s ability to generate unique ids for any column with an unique index applied. Thus, if the value of a primary key is not specified at the time an INSERT is performed, the adapter will prefetch a unique id for the primary key. This prefetching is also necessary in order to return the id after an insert.

Caveat: Operations involving LIMIT and OFFSET do not yet work!

def next_sequence_value(sequence_name)
  ary = sequence_name.split(' ')
  if (!ary[1]) then
    ary[0] =~ %r(\w+)_nonstd_seq/
    ary[0] = $1
  end
  @connection.unique_row_id(ary[0], ary[1])
end

def quote(value, column = nil)
  if value.kind_of?(String) && column && column.type == :binary
    "'#{@connection.insert_binary(value)}'"
  else
    super
  end
end
